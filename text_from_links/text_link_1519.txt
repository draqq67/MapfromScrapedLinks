Windows LAPS mit Intune und Entra ID verwalten
Windows LAPS generiert Kenn­wörter für lokale Admini­stratoren in regel­mäßigen Inter­vallen neu und speichert sie im Active Directory oder in Entra ID. Es soll somit die Ver­wendung des gleichen Admin-Pass­worts auf allen PCs ver­hindern. Seit dem kumu­lativen Update für April 2023 ist es in Windows 10 / 11 sowie in Windows Server ab 2019 enthalten.
